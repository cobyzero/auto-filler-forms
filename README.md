# Rellena automáticamente Google Forms en bucle

Este es un simple script para Chrome que te permite rellenar automáticamente formularios de Google en bucle.
Actualmente soporta Email, Edad, Genero y preguntas tipo Likert.

## Instalación

1. Descarga el archivo `manifest.json` de este repositorio.
2. Ve a `chrome://extensions/` en tu navegador de Chrome.s
3. Asegúrate de que la opción "Developer mode" esté activada.
4. Haz click en "Load unpacked" y selecciona el directorio donde guardaste el archivo `manifest.json`.
5. Ahora deberías ver la extensión "FakeFiller Forms Auto" en la lista de extensiones.

## Uso

1. Ve a una página de Google Forms.
2. El script rellenará automáticamente los formularios en bucle.
